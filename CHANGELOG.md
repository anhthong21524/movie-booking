# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-04

### Added
- Initial Nuxt 3 movie booking application scaffold
- Tailwind CSS, TypeScript strict mode, Pinia, ESLint, and Prettier setup
- Core routes for movies, booking, checkout, tickets, and admin placeholders
- Shared types, API service wrapper, auth middleware stub, and minimal state stores
- English and Vietnamese UI support with Vietnamese as the default locale
- Vercel deployment setup and repository workflow documentation
